INTRODUCTION                                          *dev_go2go-introduction*

*dev_go2go* is a plugin to support dev.go2go for Vim.


Requirements:
- Vim (|+terminal|)
- Go
- Git

Optionals:
- |vim-lsp|
- |vim-quickrun|

Latest version:
https://github.com/Jagua/vim-dev_go2go
